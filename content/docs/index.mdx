# MiniVault Documentation

Welcome to the MiniVault documentation. This guide will help you understand and configure your project management dashboard.

## What is MiniVault?

MiniVault is a unified project management dashboard that integrates:

- **Notion** - For tasks, goals, metrics, and documentation
- **Google Drive** - For file management
- **GitHub** - For repository tracking
- **Shopify** - For sales analytics (optional)

## Features

### Dashboard Sections

| Section | Description |
|---------|-------------|
| Orders | Track order fulfillment status |
| Goals | Monitor output metrics (sales, subscribers) |
| Metrics | Track input metrics (posts, interactions) |
| Sales Analytics | Revenue and sales tracking |
| Web Analytics | Traffic and conversion data |
| Tasks | One-time task management |
| Recurring Tasks | Repeating task tracking |

## Architecture

MiniVault follows a **one repository = one project** architecture. Each deployment is dedicated to a single project, configured via environment variables.

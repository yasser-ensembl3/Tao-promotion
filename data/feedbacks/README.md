# Feedback Files

This directory contains user feedback saved as text files.

## File Format

Each feedback is saved with the following naming convention:
```
YYYY-MM-DD_sanitized-title.txt
```

## File Contents

Each file contains:
- Title: The feedback title
- Date: When the feedback was submitted
- User: Name of the user who submitted the feedback
- Feedback: The actual feedback content
- Created: ISO timestamp of when the file was created

## Example

```
Title: Feature request for dark mode
Date: 2025-01-04
User: John Doe

Feedback:
Would love to see a dark mode option for the dashboard.

---
Created: 2025-01-04T15:30:00.000Z
```

## Note

These files are automatically generated by the User Feedback section of the MiniVault dashboard. They are excluded from git version control to protect user privacy.
